# Orchard Collection v4.0.2 — New Home Screen Icon

This build uses the newly approved Orchard Collection icon: the cream **O**
inside the gold ring with the simplified leaf branch on the left.

## Deploy

1. Upload the full contents of this ZIP to the existing Cloudflare Pages project.
2. Wait until the deployment shows **Success**.
3. On iPhone, remove the current Orchard Collection Home Screen app.
4. Go to **Settings → Safari → Clear History and Website Data**.
5. Reopen Safari and visit `https://app.orchardcollection.ca`.
6. Tap **Share → Add to Home Screen**.

This release version-stamps the icon URLs and advances the service-worker cache
to force iOS and browsers to download the replacement artwork.

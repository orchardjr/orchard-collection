# Orchard Collection Professional v3.0 — Foundation

## Included
- Professional desktop sidebar and improved mobile navigation
- Dedicated Label Center navigation item
- Persistent light and dark themes
- Command palette with plant search and quick actions
- Responsive workspace layout
- Updated premium visual system
- Modular source-directory foundation for future releases
- Existing Supabase data, photos, care queues, QR labels, and editing remain intact

## Install
No Supabase migration is required.

Replace the deployed project files with this package, commit, and let Cloudflare Pages redeploy. Then fully close and reopen the installed Home Screen app.

## Command palette
- Desktop: Command/Ctrl + K
- Mobile: tap Search in the header or Command Palette in the sidebar

# Orchard Collection v4.0.3 — Final App Icon

This package contains the approved final Home Screen icon:
a clean cream **O**, thin gold ring, simplified green leaves, and two gold
sparkles on a deep green background.

## Upload

1. Extract this ZIP.
2. Upload the complete contents to the existing Cloudflare Pages project.
3. Wait for the production deployment to show **Success**.

## Refresh the iPhone icon

1. Remove the existing Orchard Collection icon from the Home Screen.
2. Open **Settings → Safari → Clear History and Website Data**.
3. Fully close Safari.
4. Reopen Safari and visit `https://app.orchardcollection.ca`.
5. Tap **Share → Add to Home Screen**.

This build uses versioned icon URLs and a new service-worker cache name to
prevent iOS from reusing the old icon.

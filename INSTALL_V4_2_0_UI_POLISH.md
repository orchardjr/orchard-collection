# Orchard Collection v4.2.0 — Interface Polish

This release begins the next phase: making the application feel consistent,
native, and finished across iPhone and desktop.

## Included

- New native-style mobile app header with current-screen title and search
- Smoother page and bottom-sheet transitions
- Improved tap, focus, and hover feedback
- Refined collection photo cards with clearer text overlays
- Two-column mobile collection grid for faster browsing
- Sticky mobile search and sorting controls
- Cleaner, calmer bottom navigation with a visible active state
- More native mobile modal sheets with a drag-handle treatment
- Improved keyboard-focus visibility and reduced-motion support
- Existing plants, photos, Supabase data, Label Center, branding, and fixes preserved

## Deploy

Upload the complete ZIP contents to the existing Cloudflare Pages project.
After deployment, fully close and reopen the installed app. Open the app once in
Safari and refresh if an older cached interface remains.

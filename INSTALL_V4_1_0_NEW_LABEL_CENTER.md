# Orchard Collection v4.1.0 — New Label Center

The former desktop-style Label Studio has been replaced in the normal workflow
with a clean, mobile-first Label Center.

## New experience

- Simple Label Center home screen
- Large **Create new label** action
- Quick Label and Batch Label options
- Visual template cards
- Recent-label history with one-tap reprinting
- Plant picker that automatically fills plant information
- Full-screen mobile preview
- Simple template selector
- Save PNG and Print Label actions
- Direct **Print label** button on plant profiles now opens the new preview
- Existing professional print rendering and batch-print engine are preserved

The advanced drag-and-drop editor remains in the codebase for future v5 work,
but it is no longer shown in the standard label workflow.

## Deploy

Upload the complete ZIP contents to the existing Cloudflare Pages project.
After deployment, close and reopen the installed app. If the old Label Studio
appears, open `app.orchardcollection.ca` in Safari once and refresh.

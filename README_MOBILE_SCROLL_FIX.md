# Orchard Collection v2.4 — Mobile Edit Form Scroll Fix

This patch fixes long edit forms on iPhone.

## Install

Upload only `styles.css` to the root of the GitHub repository and replace the existing file.

Commit the change and wait for Cloudflare Pages to redeploy.

## Fixes

- Long modals now scroll vertically
- iPhone momentum scrolling is enabled
- The modal respects the top and bottom safe areas
- The close button/header remains accessible
- The save and cancel controls remain reachable at the bottom

# Orchard Collection Professional v3.1 — Polished Mobile Release

No Supabase migration is required.

## Improvements
- Compact iPhone bottom navigation with safe-area support
- Smaller, denser quick-care actions
- Two-column Care Snapshot on phones
- Reduced card height and excess spacing
- Rebuilt mobile Label Studio
- Full-height canvas workspace
- Slide-up Elements and Properties panels
- Dedicated mobile Label Studio dock
- Correct iPhone top and bottom safe areas
- Cleaner header actions and larger usable label preview
- Tap an element to open its properties automatically

## Install
Replace the deployed files with this package, commit, and wait for Cloudflare Pages to redeploy.

Because this release changes the service-worker cache, fully close the installed app after deployment and reopen it. If the old layout remains, remove the Home Screen app once and add it again from Safari.

# Orchard Collection Professional v3.1 Alpha — Label Studio

## Install
No Supabase migration is required. Replace the deployed project files with this release and redeploy Cloudflare Pages. Fully close and reopen the installed PWA after deployment.

## Included
- QL-820NWB 300 dpi printer profile
- 62 mm continuous-roll workflow
- Professional Label Center dashboard
- Four built-in templates
- Full-screen visual label studio
- Drag and resize label elements
- Text variables, QR codes, lines, and boxes
- Typography and alignment controls
- Undo and redo
- Save custom templates locally
- High-resolution PNG export
- System print / AirPrint workflow
- Multi-page batch printing

## Printer setup
Connect the QL-820NWB to the same Wi-Fi network as the iPhone, iPad, or computer. In Label Studio choose Print, then select the printer in the system print dialog. Install a 62 mm continuous roll for the included templates.
